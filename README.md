# git-tidbit
Git a tidbit everytime you commit to a repo

## Dependencies


## Setup
Clone the repo

```sh
git clone https://github.com/savala/git-tidbit.git
```

Add tidbit as an alias for git

```sh
git config --global alias.tidbit '!sh <path/to/git-tidbit>/tidbit.sh'
```

## Usage

```sh
git tidbit <message>
```
